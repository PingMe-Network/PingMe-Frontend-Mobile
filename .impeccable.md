## Design Context

### Users
- Vietnamese end users using PingMe for everyday social messaging on mobile.
- Primary context is fast, high-frequency chat flows: checking message previews, opening rooms, managing friends/invites, and starting conversations quickly.
- The UI should optimize scanability and one-thumb actions for repeated daily use.

### Brand Personality
- Friendly, modern, energetic.
- Emotional goals: feel clear, responsive, and socially warm without visual clutter.
- Core tone is practical-first with a polished, trustworthy finish.

### Aesthetic Direction
- Light-first interface with pink brand accents (`#DF40A3`) and neutral card surfaces.
- Web-aligned look: clean list surfaces, rounded containers, compact metadata, clear status badges, and subtle emphasis for key actions.
- Avoid noisy gradients, decorative effects, or overly playful iconography in primary workflows.

### Design Principles
1. Prioritize readability first: names, previews, and action labels must be instantly scannable.
2. Keep hierarchy obvious: primary actions use solid accent; secondary actions use soft neutral or tinted surfaces.
3. Use consistent geometry: rounded cards/pills and predictable spacing rhythm across messages and contacts.
4. Prefer professional iconography (Lucide style) with clear semantic color cues.
5. Preserve functional density: improve visual quality without increasing interaction steps.
